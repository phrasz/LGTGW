Command Prompt 101:
===================
0. Start Cmd <Press Windows Button + R; Type: CMD; Press Enter>
1. Type: CD C:\
2. Make folders as needed by typing: mkdir _______

Example:
0. Find a Folder to work out of <C:\My Programming\LPTHW\>
